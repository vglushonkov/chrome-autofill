Test Form for Fields Autofill Extension

How to test the extension:

  • Right-click on any input field below
  • Select "Autofill" from the context menu
  • Enter a value and save it
  • Notice that the extension icon in the browser toolbar now shows a badge with the number of filled fields
  • Refresh the page to see the fields auto-fill
  • Switch to other tabs to see the badge disappear/reappear based on filled fields
