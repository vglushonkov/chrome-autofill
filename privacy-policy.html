Privacy Policy

Fields Autofill Extension

Last Updated: July 24, 2025

TL;DR: This extension does NOT collect, store, or transmit any of your personal data. Everything stays on your device.

1. Data Collection

The Fields Autofill extension:

  • Does NOT collect any personal information
  • Does NOT send any data to external servers
  • Does NOT track your browsing activity
  • Does NOT use analytics or monitoring services

2. Data Storage

All data saved by this extension is stored locally on your device using Chrome's secure storage API:

  • Form field values you choose to save
  • Website-specific preferences
  • Extension settings
Important: This data never leaves your browser and is only accessible by the extension itself.

3. Data Usage

The saved data is used exclusively for:

  • Automatically filling form fields when you visit websites
  • Displaying saved values in the extension popup
  • Managing your autofill preferences

4. Data Security

Your data is protected by:

  • Chrome's built-in security mechanisms
  • Local-only storage (no cloud synchronization)
  • No external network connections
  • Same-origin policy enforcement

5. Data Control

You have complete control over your data:

  • View all saved data through the extension popup
  • Edit or delete individual field values
  • Clear all data for specific websites
  • Completely remove all extension data

6. Permissions Explanation

The extension requests the following permissions:

  • "activeTab": To read and fill form fields on the current page only
  • "contextMenus": To add the "Autofill" option to right-click menus
  • "storage": To save your form field values locally
  • "<all_urls>": To work on all websites (data stays local)
Note: Despite the "<all_urls>" permission, the extension never sends any data to external servers.

7. Third-Party Services

This extension does not use any third-party services, analytics, or external APIs.

8. Children's Privacy

This extension does not knowingly collect any information from children under 13 years of age.

9. Changes to Privacy Policy

Any changes to this privacy policy will be reflected in the extension's store listing and in updated versions of this document.

10. Data Removal

To completely remove all data:

  • Open the extension popup
  • Go to "All Sites" tab
  • Click "Clear All Data"
  • Or uninstall the extension entirely

Contact Information

If you have any questions about this privacy policy or the extension's data practices, you can:

  • Leave a review on the Chrome Web Store
  • Report issues through the extension's support channel
